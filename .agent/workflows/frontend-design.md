---

## 1. Define Aesthetic Direction / 确定审美方向

- Choose between **Bold Maximalism** or **Refined Minimalism**.
  选择 **大胆极繁主义** 或 **精致极简主义**。
- Do not mix; be intentional with the visual language.
  不要混用；视觉语言必须明确、有意识地选择。

---

## 2. Typography Selection / 字体选择

- [ ] Check if using generic fonts (e.g., Inter, Arial). If so, consider distinct display fonts for headers.
      检查是否使用了通用字体（如 Inter、Arial）。如果是，考虑为标题使用独特的展示字体。
- [ ] Ensure typography hierarchy is clear and deliberate.
      确保排版层级清晰且刻意设计。

---

## 3. Color & Theme Setup / 配色与主题设置

- [ ] Define CSS variables for consistency.
      定义 CSS 变量以保持一致性。
- [ ] Ensure a dominant color is paired with sharp accents.
      确保主色调搭配鲜明的强调色。
- [ ] Avoid "default" AI color palettes (e.g., standard generic gradients).
      避免"默认"AI 配色方案（如常见的通用渐变）。

---

## 4. Motion & Interaction Planning / 动效与交互规划

- [ ] Identify high-impact moments for CSS animations (e.g., hero entry).
      识别需要 CSS 动画的高影响力时刻（如首屏进入动画）。
- [ ] Add micro-interactions (hover, active states) to all interactive elements to avoid a "static" feel.
      为所有交互元素添加微交互（悬停、激活状态），避免"静态"感。

---

## 5. Implementation & Review / 实施与审查

- [ ] Write production-grade HTML/CSS.
      编写生产级别的 HTML/CSS。
- [ ] Verify responsiveness across breakpoints.
      验证在各个断点下的响应式表现。
- [ ] Check against common pitfalls: cookie-cutter layouts, lack of depth/texture, or inconsistent spacing.
      检查常见陷阱：千篇一律的布局、缺乏深度/纹理、或不一致的间距。
